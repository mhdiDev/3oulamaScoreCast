Structure des écrans — mobile-first, FR par défaut

AUTH (public)
  • /invite/[token] — page d'inscription via lien
  • /login — connexion email/mdp
  • /forgot-password — reset par email
  • /reset-password/[token]
APP (authentifié)
  • / — Dashboard : prochains matchs + mon classement
  • /matches — Calendrier complet par phase
  • /matches/[id] — Détail match + saisie pronostic
  • /leaderboard — Classement général + par journée
  • /profile — Avatar, pseudo, préfs notifs
  • /rules — Barème et règles du jeu
ADMIN
  • /admin — Dashboard admin
  • /admin/invitations — Générer/révoquer liens
  • /admin/users — Liste users, désactiver compte
  • /admin/scoring — Configurer barème
  • /admin/matches — Forcer correction d'un score
