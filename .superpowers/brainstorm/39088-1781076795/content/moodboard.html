Guidelines clés :
• Focus visible sur tous les éléments interactifs (outline 2px #3b82f6, offset 2px)
• Touch targets min 44×44px sur mobile
• Aria-labels sur tous les boutons icône et compteurs de score
• role="status" sur les mises à jour dynamiques (scores, classement)
• Skip-to-content link en haut de chaque page
• Texte alternatif sur tous les drapeaux (alt="Drapeau France")
• Couleur jamais seule pour transmettre une information (+ icône ou texte)
